How It Works

​

1. Write

​

Write your UI in .wf files using the declarative syntax. Components, pages, stores, and styles — all in one language.

​
​
​
​

2. Build

​

The Rust compiler lexes, parses, lints for accessibility, and generates optimized HTML + CSS + JS output.

​
​
​
​

3. Ship

​

Deploy the static output anywhere — GitHub Pages, Netlify, Vercel, or any static hosting. No server required.
