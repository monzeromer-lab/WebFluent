The t() Function

вЂ‹

Use t() to look up translated text. It is reactive вЂ” all t() calls update when the locale changes.

вЂ‹

вЂ‹

Fallback Behavior

вЂ‹

If a key is missing in the current locale:

вЂ‹

1. Falls back to the defaultLocale translation

вЂ‹

2. If still missing, returns the key itself (e.g., "nav.home")

вЂ‹
вЂ‹

SSG + i18n

вЂ‹

When both SSG and i18n are enabled, pages are pre-rendered with the default locale text. After JavaScript loads, locale switching works normally.
