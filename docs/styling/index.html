Themes

​

4 built-in themes. Set in webfluent.app.json. Each preview below shows the actual colors and style of that theme.

​
default​

Default

​
​

Clean, modern light theme with blue primary.

​
​ ​ Tag​
​
​
"theme": { "name": "default" }​
​
​
dark​

Dark

​
​

Dark backgrounds with light text and vibrant accents.

​
​ ​ Live​
​
​
"theme": { "name": "dark" }​
​
​
minimal​

Minimal

​
​

Black and white. No shadows, no border-radius. Pure content.

​
​ Note​
​
​
"theme": { "name": "minimal" }​
​
​
brutalist​

Brutalist

​
​

Monospace font, bold red primary, hard offset shadows.

​
​ Alert​
​
​
"theme": { "name": "brutalist" }​
​
​
​

Custom Tokens

​

Override any design token in your config to customize the theme.

​
{ "theme": { "name": "default", "tokens": { "color-primary": "#8B5CF6", "color-secondary": "#EC4899", "font-family": "Poppins, sans-serif", "radius-md": "1rem" } } }​
​
​

Style Blocks

​

Override styles on any component with inline style blocks.

​
Button("Custom") { style { background: "#8B5CF6" padding: xl radius: lg } }​
