CLI Reference

​

The WebFluent command-line interface.

​

wf init

​

Create a new WebFluent project.

​
wf init <name> [--template spa|static|pdf]​
​
​
​ ​ ​ ​ ​ ​ ​ ​ ​
ArgumentDescription
nameProject name (creates a directory)
--template, -tTemplate: spa (default), static, or pdf
​

SPA: interactive app with routing and state. Static: SSG site with i18n. PDF: document generation with tables, headings, and auto page breaks.

​

wf build

​

Compile .wf files to HTML, CSS, and JavaScript.

​
wf build [--dir DIR]​
​
​
​ ​ ​ ​ ​ ​
OptionDescription
--dir, -dProject directory (default: current directory)
​

The build pipeline: Lex all .wf files, parse to AST, run accessibility linter, generate HTML + CSS + JS, write to output directory.

​

Output depends on config:

​

SPA mode (default): single index.html + app.js + styles.css

​

SSG mode (ssg: true): one HTML per page + app.js + styles.css

​

PDF mode (output_type: pdf): a single .pdf file

​
​

wf serve

​

Start a development server that serves the built output.

​
wf serve [--dir DIR]​
​
​

Serves files from the build directory. SPA fallback: all routes serve index.html so client-side routing works. Port is configured in webfluent.app.json (default: 3000).

​

wf generate

​

Scaffold a new page, component, or store.

​
wf generate <kind> <name> [--dir DIR]​
​
​
​ ​ ​ ​ ​ ​ ​ ​ ​ ​ ​ ​ ​ ​ ​ ​
KindCreatesExample
pagesrc/pages/Name.wfwf generate page About
componentsrc/components/Name.wfwf generate component Header
storesrc/stores/name.wfwf generate store CartStore
​

Configuration

​

All config is in webfluent.app.json at the project root.

​
{ "name": "My App", "version": "1.0.0", "author": "Your Name", "theme": { "name": "default", "mode": "light", "tokens": { "color-primary": "#6366F1" } }, "build": { "output": "./build", "minify": true, "ssg": false, "output_type": "spa", "pdf": { "page_size": "A4", "default_font": "Helvetica", "output_filename": "report.pdf" } }, "dev": { "port": 3000 }, "meta": { "title": "My App", "description": "Built with WebFluent", "lang": "en" }, "i18n": { "defaultLocale": "en", "locales": ["en", "ar"], "dir": "src/translations" } }​
