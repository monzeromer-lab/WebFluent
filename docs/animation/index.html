​

Toggle the switch to see enter/exit animations on the card below.

​
​
if showCard, animate(scaleIn, fadeOut) { Card(elevated) { Text("Animated content") } }​

Conditional Animations

​

Attach enter and exit animations to if blocks.

​
if visible, animate(slideUp, fadeOut) { Card { Text("Appears with slideUp, exits with fadeOut") } } if expanded, animate(scaleIn, scaleOut) { Text("Scales in and out") }​
​
​

List Stagger

​

Animate list items with staggered delays.

​
for item in items, animate(slideUp, fadeOut, stagger: "50ms") { Card { Text(item.name) } }​
​
​

Transition Blocks

​

Smooth CSS transitions on property changes.

​
Button("Hover me") { transition { background 200ms ease transform 150ms spring } }​
